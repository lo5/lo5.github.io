Nitro v0.9 shipped with CLI

10 Jun 2022

Screencast: https://twitter.com/CrunchingData/status/1535346872520306688

Nitro v0.9 now has a CLI that lets you nitro run a URL pointing to a #Python file or gist on Github or anywhere else. Makes it easy to run examples: just copy/paste commands from the docs into your terminal.
